# Кіру мәселесін тексеру

## Мәселені шешу қадамдары:

### 1. Пайдаланушыларды тексеру
```bash
# Базада пайдаланушылар бар ма тексеру
npm run db:seed
```

### 2. Серверді қайта іске қосу
```bash
# Серверді тоқтатып, қайта іске қосу
npm start
# немесе
npm run dev
```

### 3. Кіру тестілеу

Браузерде `/login` бетіне барып, мына пайдаланушылармен кіріңіз:

- **admin** / **admin123**
- **engineer** / **engineer123**  
- **operator** / **operator123**

### 4. Мәселе болса, тексеру:

1. **Браузер консольінде қателерді тексеру** (F12)
2. **Сервер логтарын тексеру** - login әрекеттерін бақылау
3. **Cookie-ларды тексеру** - `m-admin.sid` cookie-сы сақталған ба?

### 5. Егер пайдаланушылар жоқ болса:

```bash
# Seed файлын қайта орындау
npm run db:seed
```

### 6. Егер сессия сақталмаса:

- Браузер cookie-ларын тазалау
- Браузерді қайта іске қосу
- Серверді қайта іске қосу

## Қосылған жақсартулар:

1. ✅ Сессия конфигурациясына `sameSite: 'lax'` қосылды
2. ✅ Сессияны сақтау кезінде `req.session.save()` қосылды
3. ✅ Login маршрутына логирование қосылды
4. ✅ Пайдаланушылар seed файлы арқылы қосылды
