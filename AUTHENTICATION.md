# Аутентификация жүйесі

## Қосылған мүмкіндіктер

1. **3 рөл қосылды:**
   - `admin` - Админ
   - `engineer-technologist` - Инженер-технолог
   - `operator` - Оператор

2. **Логин/парольмен кіру жүйесі**

## Пайдаланушылар

Бастапқы пайдаланушылар (seed файлы арқылы қосылады):

| Пайдаланушы аты | Пароль | Рөл |
|----------------|--------|-----|
| admin | admin123 | Админ |
| engineer | engineer123 | Инженер-технолог |
| operator | operator123 | Оператор |

## Орнату

### 1. Пакеттерді орнату
```bash
npm install
```

### 2. База деректерін синхрондау
```bash
npm run db:migrate
```

### 3. Бастапқы пайдаланушыларды қосу
```bash
npm run db:seed
```

### 4. Серверді іске қосу
```bash
npm start
# немесе
npm run dev
```

## API Endpoints

### Аутентификация

- `POST /auth/login` - Кіру
  ```json
  {
    "username": "admin",
    "password": "admin123"
  }
  ```

- `POST /auth/logout` - Шығу (аутентификация қажет)

- `GET /auth/me` - Ағымдағы пайдаланушыны алу (аутентификация қажет)

### Пайдаланушыларды басқару (тек админ)

- `GET /auth/users` - Барлық пайдаланушыларды алу
- `POST /auth/users` - Жаңа пайдаланушыны қосу
  ```json
  {
    "username": "newuser",
    "password": "password123",
    "role": "operator",
    "fullName": "Толық аты"
  }
  ```

## Қорғалған маршруттар

Барлық `/api/*` маршруттары аутентификациямен қорғалған. Кірусіз пайдаланушылар `/login` бетіне бағытталады.

## Рөлдер

- **admin**: Барлық мүмкіндіктерге қол жеткізу, пайдаланушыларды басқару
- **engineer-technologist**: Технологиялық процестерді басқару
- **operator**: Операцияларды орындау
